1.面向对象的三大特性
    封装
        成员私有化
        在类中定义的成员，只能在类中被访问，类的外部访问不到
        有些类中的成员不想被子类继承，也可以进行私有化

        如果成员以双下划线开头，名字会被改写，改为_类名__成员名

    继承
        一个类可以从另一个类中，去继承非私有属性和方法
        被继承的类叫做父类，也叫基类
        继承的类叫做子类，也叫派生类
        子类可以继承父类的非私有成员，当然子类也可以定义自己特有的方法，也可以对父类的方法进行重写
        语法：
            class 父类:
                pass
            class 子类(父类):
                pass
        多继承：一个子类可以同时继承多个父类
        多继承语法:  class 子类(父类1, 父类2)
            调用方法时先在子类中查找，若不存在则从左到右依次查找多个父类中是否包含方法
        可以通过super()访问父类中的属性和方法,也可以通过Person.方法(self)
        方法的重写：
            如果在父类和子类中存在相同的方法，子类的方法会将父类的方法覆盖掉，这种情况叫方法的重写或者叫方法的覆盖

    多态
        一个对象的多种形态
            多态应用在方法的返回值位置
            多态应用在方法参数位置
