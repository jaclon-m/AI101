
1.容器数据类型
    字符串 str
        str是不可变的，不能直接对原字符串进行修改，  如果修改会创建新的对象
        str是有序的，字符串中的每个字符都有对应的下标(索引)，可以通过索引获取字符以及对str进行切片操作
        str的定义方式
            单引号
            双引号
            三引号

    列表 list
        list是可变的，可以对list中的元素进行修改
        list中元素可以重复
        list是有序的，每个元素都有对应的下标(索引)，可以通过索引获取元素以及对list进行切片操作
        list中可以存放不同类型的元素
        使用[]定义list

    元组 tuple
        tuple是不可变的，不能直接对元组中的元素进行修改
        tuple中元素是可以重复的
        tuple是有序的，每个元素都有对应的下标（索引），可以通过索引获取元组中的元组以及对元组进行切片
        tuple中可以存放不同类型的数据
        使用()定义tuple

    集合 set
        set是可变的，【可以对set中元素进行修改】，可以向set中添加元组以及从set中删除元素
        set中元素不能重复
        set是无序的，set中的元素没有对应的下标，不能通过下标获取元素，当然也不能进行切片
        set中可以存放不同类型的数据
        set定义{}  set()

    字典 dict
        以k-v键值对的形式存储数据的  key:value
        其中key部分内容是不可变的      数值、字符串、元组
        其中value部分内容是可变的     可以存放不同类型的数据
        dict是无序的，不能通过下标获取元素，不能进行切片操作
        dict的key是唯一的，不能重复；value可以重复
        dict使用{}定义，键和值之间用:分隔，每一个键值对之间用,分隔

2.各种容器类型对比
    数据结构	                    是否可变	        是否重复	        是否有序	                        定义符号
    列表（List）	                可变	            允许	            有序	                            []或list()
    元组（Tuple）	            不可变	        允许	            有序	                            ()或tuple()
    字典（Dictionary）	        k:不可，v:可	    键不，值行	    键无序	                        {}或dict()
                                                              （Python 3.7+版本中保持插入顺序）
    集合（Set）	                可变	            不允许	        无序	                            {}或set()
